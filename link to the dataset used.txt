Here I have used this "Fabric Defects Object Detection Dataset" from FigShare:
Dataset link : https://figshare.com/articles/dataset/FD_Dataset_7z/25546465

It contains annotated images representing four fabric defect classes:

Holes
Stains
Cuts
Tear Cuts
